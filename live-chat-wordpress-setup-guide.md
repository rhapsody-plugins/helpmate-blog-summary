# Live Chat on WordPress: Zero-Code Installation Guide\n\nLive chat delivers 73% customer satisfaction and ~20% conversion lift according to Nextiva 2025 research. This guide shows how to add Helpmate Live Chat without touching theme files or writing code.\n\n## Installation\n\n- Install via WordPress Plugins > Add New, search \"Helpmate\" or upload the zip\n- Activate, run the onboarding wizard—API key generates automatically\n- Helpmate injects through wp_footer hook; no header.php edits required\n- Widget appears bottom-right by default; verify in incognito window\n\n## Widget Placement & Performance\n\n- Appearance tab controls positioning: all pages, specific post types, or URL exclusions\n- Bottom-right avoids navigation conflicts; auto-sizes mobile (<768px)\n- Enable lazy loading in Behavior tab to delay initialization until scroll/interaction\n- Use script deferral instead of async to maintain execution order\n- With optimizations, chat impact stays under 100ms on Core Web Vitals\n\n## Team Access\n\n- Go to Helpmate > Control Center > Teams & Roles\n- Three levels:\n  - Administrator: Full settings, analytics, team management\n  - Agent: Live queue access, assigned conversations\n  - Viewer: Read-only analytics and history\n- Assign by job duties, not titles; marketing can have Viewer access without queue clutter\n\n## AI-to-Human Handover\n\n- Configure triggers in Behavior > Chatbot Settings:\n  - Keywords: \"refund\", \"complaint\", \"speak to human\"\n  - Three consecutive AI responses without resolution\n  - Explicit human request button\n  - Outside business hours when agents available\n- Knowledge Base trains AI on products, posts, pages, URLs, Q&A pairs, CSVs, PDFs\n- Handover delivers full context including AI interaction history\n\n## Performance Settings\n\n- Enable script deferral in Behavior tab\n- Adjust connection intervals: 3s during active chats, 30s idle\n- Exclude Helpmate dynamic endpoints from page caching (WP Rocket, W3 Total Cache, LiteSpeed)\n- Test via sandbox in admin sidebar before going live\n\n## Troubleshooting\n\n- Widget missing? Check plugin active, API key valid, theme implements wp_footer()\n- JavaScript errors? Exclude Helpmate scripts from aggressive minification plugins\n- Message delivery issues? Verify wp-cron.php runs periodically\n- Conversation data stays in WordPress DB unless explicitly deleted during uninstall\n\n## Requirements\n\n- PHP 7.4+, WordPress 5.8+, HTTPS active\n- Free tier includes live chat, AI chatbot, CRM features\n- Unlimited agents on free plan with role controls\n\nKey success pattern: build Knowledge Base first, define team roles before opening queue, test thoroughly in sandbox.
